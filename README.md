# Employee_Example
The Application is developed in Node.js Employee Example - Node API Service only for create update remove and get all

This application is for the wireframe 3 as per section 1 

## Pre-requisites


To get started, 
you'll need to have the following requirements installed

-
- Node.js<sup>1</sup>

- npm

- MongoDB 2.6.x / 3.2.x<sup>2</sup>


## Getting started

# Ensure `mongod` is running, either as a service or in another shell
	
-in node_module start npm
-then run node server.js

-you can run this on localhost port no 3000.
